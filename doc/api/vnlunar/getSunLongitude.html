int getSunLongitude(
  1. int dayNumber,
  2. int timeZone
)

Compute sun position at midnight of the day with the given Julian day number. The time zone if the time difference between local time and UTC: 7.0 for UTC+7:00. The function returns a number between 0 and 11. From the day after March equinox and the 1st major term after March equinox, 0 is returned. After that, return 1, 2, 3 ...

Implementation

int getSunLongitude(int dayNumber, int timeZone) {
  return (sunLongitude(dayNumber - 0.5 - timeZone / 24) / pi * 6).floor();
}
